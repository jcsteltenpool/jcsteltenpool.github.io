Over mij

Hi, ik ben Joost! Ik ben * jaar oud en sinds jaar ben ik actief als front-end developer. In dit werk komt alles samen waar ik van hou: creatief bezig zijn, oplossingen bedenken voor problemen en net zolang knutselen en puzzelen tot iets werkt of oogt zoals een opdrachtgever en ik het willen!

In het verleden heb ik gewerkt als theatermaker, acteur en muzikant/geluidsontwerper. Daarnaast heb ik een aantal jaar gewerkt in de marketing en communicatie en het onderwijs. Alle kennis en vaardigheden die ik heb opgedaan, gebruik ik in mijn werk als developer. Niet alleen mijn creativiteit, maar ook mijn brede ervaring met user experience.

Projecten

Theaterbazen

elementor // css // javascript

Theaterbazen is een Amersfoorts project van NEOS Cultuuronderwijs en Stadsprogrammering033 om zoveel mogelijk kinderen in Amersfoort en daarbuiten enthousiast te maken voor (jeugd)theater. De site is gebouwd in Wordpress Elementor en waar nodig aangevuld met custom html, css en javascript.

1000 Bommen & Granaten

html // css flexbox // css grid // javascript

Dit scorebord voor het gelijknamige dobbelspel van 999 Games is volledig responsief door het gebruik van css flexbox en css grid. Met behulp van DOM event manipulatie, functions en array iterators kunnen spelers worden toegevoegd, namen aan spelers worden toegewezen en de score worden gemanipuleerd.

How long has it been?

html // css flexbox // javascript

Dit programma berekent hoeveel tijd in hele jaren, maanden en dagen er is verstreken tussen de opgegeven datum en vandaag. Hierbij wordt in javascript gebruik gemaakt van date objects, for loops, if else statements en switch statements.

De Skate Centrale

html // css flexbox // javascript

Deze dummy site voor de Skate Centrale in Amersfoort maakt gebruik van css flexbox en responsief design. De mobiele versie bevat een door javascript aangestuurd drop-down menu. Javascript omvat DOM event manipulatie in de vorm van event listeners en query selectors.
